    Chainable request modifiers. These define the available modifiers that help with the configuring a request to Akamai. Each modifier will return an altered function with a new set of modifiers. This function, when called, will pass the configured options.

    exports.MODIFIERS = {
  production: {domain: 'production'},
  staging: {domain: 'staging'},
  cpcode: {type: 'cpcode'},
  url: {type: 'arl'},
  uri: {type: 'arl'},
  arl: {type: 'arl'},
  remove: {action: 'remove'},
  invalidate: {action: 'invalidate'}
};
